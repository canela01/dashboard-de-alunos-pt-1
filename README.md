# dashboard-de-alunos-pt-1

🧠 Conceitos Práticos Praticados
🛠️ Estrutura Arquitetural (HTML5)
Divisão de Layout: Uso de tags semânticas como <aside> para o menu de navegação lateral, <main> para o conteúdo do painel e <section> para os blocos de dados.

Componentização Semântica: Uso de tabelas estruturadas (<table>, <thead>, <tbody>), listas de ações e elementos de controle de formulários (<select>, <input>, <button>).

🎨 Design e Layout Avançado (CSS3)
Variáveis CSS (Custom Properties): Centralização da paleta de cores em variáveis (--bg-primary, --text-color) para facilitar a implementação do modo noturno (Dark Mode).

Posicionamento e Grid-Layout: Técnicas aplicadas para criar o alinhamento de colunas dos cards de métricas e o espaçamento correto entre os painéis.

Estilização de Tabelas e Estados: Customização de linhas de tabelas, efeitos :hover e cantos arredondados (border-radius) para um acabamento visual moderno e limpo.

⚡ Manipulação Dinâmica de Dados (JavaScript)
Seleção e Atualização de KPIs: Uso de document.querySelector() para interceptar os cards e atualizar os valores numéricos com base em interações lógicas.

Eventos de Controle: Criação de funções de escuta (addEventListener) para monitorar cliques de filtragem, buscas textuais ou o botão de alternância de tema.

Manipulação de Classes: Uso do .classList.toggle() na tag <body> ou em containers para disparar a troca de estilos visuais de forma instantânea.

Lógica de Arrays e Loops: Manipulação e renderização de listas de dados dinâmicos utilizando estruturas de repetição e funções nativas do JavaScript.
